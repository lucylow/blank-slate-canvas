API ENDPOINTS:
- GET  /                                     - Root endpoint
- GET  /health                               - Health check
- GET  /api/tracks                           - List all tracks
- GET  /api/tracks/{track_id}                - Get track details
- GET  /api/tracks/{track_id}/races/{race}   - Get race info
- GET  /api/dashboard/live                   - Main dashboard (AI analytics)
- POST /api/analytics/tire-wear              - Tire wear prediction
- POST /api/analytics/performance            - Performance analysis
- POST /api/analytics/strategy               - Strategy optimization
- GET  /api/analytics/gap-analysis           - Gap calculations

DEPLOYMENT:
1. Extract: tar -xzf pitwall-backend.tar.gz
2. Install: pip3 install -r requirements.txt
3. Run: python3.11 -m uvicorn app.main:app --host 0.0.0.0 --port 8000
4. Access: http://localhost:8000/docs
